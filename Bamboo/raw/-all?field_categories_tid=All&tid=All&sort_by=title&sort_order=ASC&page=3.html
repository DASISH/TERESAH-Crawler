iPhoto

iPhoto can download images from your camera, and allows you to do basic image touch-up, tag people, present slideshows, and share photos in a variety of ways (email, Facebook, Flickr, etc.). It is available in the Mac App Store as part of iLife.
IrfanView

An image viewer and converter with simple editing features; can also generate slideshows and save them as .exe files. Includes support for Photoshop filters.
iShowU

iShowU is publishing and recording software for Mac. It enables you to record audio and video on your screen, capture full screen shots, and program recording settings.
iSpring

iSpring is an authoring tool that allows you to create online presentations with slides synced with audio and video narration. You can create quizzes and courses, add attachments and web links, and integrate these courses with BlackBoard LMS.
IssueLab

IssueLab is a research archive in the nonprofit sector. In addition to archiving social policy research by nonprofits at no cost, IssueLab distributes this research online with a broad audience via e-newsletters, RSS feeds, and social media.
iTunesCyrFix

iTunesCyrFix can convert jibberish text in the title, artist, artist, or other ID3 tags in iTunes (e.g. Ñïëèí) into readable UTF-8 Cyrillic text.
iWrite

Web-based service to keep track of the status of your writing submissions
JabRef

JabRef is an open source bibliographic reference management tool. You can import BibTeX and LaTeX files to generate bibliographies in a number of different styles.
Jarnal

Jarnal is an open-source application for note-taking, sketching, journaling, creating a presentation, annotation a document/PDF, and collaborate with others.

JavaScript InfoVis Toolkit enables users to create interactive web-based visualizations; the demos page has some good examples along with the underlying code.

Jekyll

Jekyll is a simple, blog aware, static site generator. It takes a template directory (representing the raw form of a website), runs it through Textile or Markdown and Liquid converters, and spits out a complete, static website suitable for serving with Apache or your favorite web server.
JGAAP (java graphical authorship attribution program)

JGAAP is software designed for textual analysis, text categorization, and authorship attribution
Jing

Jing allows you to take screenshots, record screencasts, and instantly share information.
JIRA

JIRA is project and bug tracking software that integrates with other Atlassian products like the Confluence wiki. JIRA available either for installation on a local server or via cloud hosting.
Joomla

Joomla is an open source content management system (CMS), which enables you to build Web sites and powerful online applications.
jotform

Jotform allows users to create web forms (for surveys, etc.) using a drag-and-drop interface.
Journler

Journler is a daily notebook and entry based information manager. Scholars, teachers, students, writers, and everyday users may use this on a daily basis to integrate their notebook content to other sources of media such as audio and video.
JQuery
JsFiddle

JsFiddle is a playground for web developers, a tool which may be used in many ways. One can use it as an online editor for snippets build from HTML, CSS and JavaScript. The code can then be shared with others, embedded on a blog, etc.
Jumpchart

Jumpchart is a way to collaboratively design website wireframes and sitemaps, and includes the ability to create navigation menus, draft and organize content, and track changes.
Juxta

Juxta is an open-source cross-platform tool for comparing and collating multiple witnesses to a single textual work.
Kaleidoscope 2.0

Kaleidoscope is one of the world's best tools for spotting differences in images and text, and now it supports merging of files and folders, too. Kaleidoscope integrates directly with Git, Subversion, Mercurial, and Bazaar to fit perfectly in your workflow.
Kartograph

Kartograph is a new framework for building interactive map applications without Google Maps or any other mapping service. It was created with the needs of designers and data journalists in mind.
Kartoo

Kartoo is a search engine that works across many different languages. Kartoo also provides a variety of other features and games.
Koha

Koha is an open source integrated library system (ILS) that has also been used for virtual library systems (e.g. recreating historic libraries).
KORA

KORA is an digital repository that allows institutions to ingest, manage, and deliver digital objects and metadata.
Korbo

Korbo is a powerful aggregation platform for gathering Linked Data objects relevant to your area of research into single workspaces or “baskets”.
KwMap

KwMap is a search engine that functions as a "keyword map" across the web. KwMap utilizes a keyword generator with a graphical interface that allows you to navigate from your initial query to results for related topics.
LATtice

LATtice lets you explore and compare texts across entire corpora but also allows you to “drill down” to the level of individual LATs to ask exactly what rhetorical categories make texts similar or different.
LC Newspaper Viewer

The LC Newspaper Viewer is an open-source web application that understands how to model newspaper data created according to a set of technical guidelines, with the goal of publishing an online archive like Chronicling America.
Leaflet

Leaflet is a modern open-source JavaScript library for mobile-friendly interactive maps.
Legend Maker

A simple and easy tool for creating EPUB, MOBI, and other ebook formats.
Lemon8-XML

Lemon8-XML helps authors convert scholarly papers from word processor formats (e.g. .doc, .odt) into XML-based formats

Leximancer

Leximancer is text analysis software that can create topic and concept based network visualizations and includes a sentiment analyzer.
Lexipedia

Lexipedia is a metadictionary displaying a semantic map of parts of speech related to the word of inquiry.
Lexomics

The term "lexomics" was originally coined to describe the computer-assisted detection of "words" (short sequences of bases) in genomes. When applied to literature as we do here, lexomics is the analysis of the frequency, distribution, and arrangement of words in large-scale patterns.
Lexos

Lexos is an online tool that enables you to "scrub" (clean) your text(s), cut a text(s) into various size chunks, manage chunks and chunk sets, and choose from a suite of analysis tools for investigating those texts.
Lextek

Lextec offers a range of services and software for full-text indexing search and retrieval; automatic classification, routing, and filtering electronic text according to user defined profiles.
LibraryLookup

Library Lookup is a bookmarklet that can look up whether a book is available in the user's local library, based on information about that book on a separate website (e.g. Amazon).
LibX

LibX is a Firefox or Chrome plugin that enables direct access to the user's library's resources, including support for Google Scholar, CiteULike, COinS, and xISBN.
LilyPond

LilyPond is text-based music engraving software, with specific text input conventions.
LimeService

LimeService is basically the hosted version of the GNU licensed LimeSurvey. It is a survey service-platform to prepare, run and evaluate on-line surveys. Besides basic free usage you are always getting the full feature set with no monthly fees or subscription plans.
LimeSurvey

Online web survey tool written in PHP using MySQL, MSSQL or Postgres database. Multilingual site with demo, feature list, documentation [Open Source, GPL3]

Matlab
