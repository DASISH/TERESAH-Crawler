Resource Description
Sort My List

Sort My List will alphabetically sort a list of plain text data, and do some simple text reformatting (e.g. going from CSV to TSV)
Soundflower

Soundflower is a MacOS system extension that allows applications to pass audio to other applications. Soundflower is easy to use, it simply presents itself as an audio device, allowing any audio application to send and receive audio with no other support needed.

SoundScriber

A software application for the playback of audio recordings. SoundScriber offers specific functionality for researchers that wish to transcribe a recording.
SpiderOak

SpiderOak provides an easy, secure and consolidated free online backup, sync, sharing, access & storage solution.
Springpad

Springpad allows you to create, organize and share annotations and notes. You may bookmark web pages, search online products, use tags to better organize your information, create your own digital cork-board, and share your information by making your pages "public."
Springpad

Springpad allows you to create, organize and share annotations and notes. You may bookmark web pages, search online products, use tags to better organize your information, create your own digital cork-board, and share your information by making your pages "public."
SPSS Statistics

The base SPSS Statistics package includes support for descriptive statistics, bivariate statistics, correlation, prediction for numerical outcomes, and prediction for identifying groups.
SQL Server

A server-side software application that enables databases conformant to a relational data model to be created, managed and queried. Information, most commonly text may be stored as one or more records contained within a table.
Stanford POS Tagger

The Stanford Part-of-Speech Tagger includes English, Arabic, Chinese, and German tagger modules.
Stanza

Stanza allows you to read books on your iPhone, iPod Touch and iPad.
StatCrunch

StatCrunch is web-based statistical analysis and data-sharing software.
Statistical Lab

Statistical Lab is a graphical user interface for R designed to make statistical analysis easier to understand.
Stories Matter

Stories Matter is an oral history database that allows you to cut, index, and export audio and video recordings.
Storify

Storify is a way to tell stories using social media such as Tweets, photos and videos. You search multiple social networks from one place, and then drag individual elements into your story. You can re-order the elements and also add text to give context to your readers.
StoryKit

StoryKit allows you to create an electronic storybook.
Substance

Substance is a fully web-based document authoring and publishing platform.
Subversion Version System (SVN)

Subversion Version System (SVN) is a version control system used to control and manage multiple versions of data objects. Access and revision to objects are carefully controlled, to prevent unauthorised access and alteration.

Sumo Paint

Web-based image editor with photo editing and drawing features. A free account is required.
Superfastmatch

Superfastmatch is designed to find exact duplicates of text strings between documents.
Supermemo

SuperMemo is a learning software that is devoted to increasing the speed of learning and heightening the level of knowledge retention.
Survey Daddy

Create a survey to measure and record your students' progress against curriculum outcomes. Use Likert questions to rate students based on a rubric and text questions to record qualitative data to share with parents and faculty.
Survey Monkey

Survey Monkey is a web-based survey creation and distribution site, with free and paid plans that allow users to create surveys and collect responses through a link, email, Facebook, or being embedded in a website or blog.
SwiftRiver

SwiftRiver is free and open source web-based software for real-time filtering, curation, and qualitative analysis of social media data (Twitter, etc.)
T-PEN (Transcription for Paleographical and Editorial Notation)

Extensive set of tools to allow collaborative transcription of manuscript pages in TEI-compliant XML.

Features of T-PEN through version 1.2 [from project blog]
Tabula

Tabula lets you upload a (text-based) PDF file into a simple web interface and magically pull tabular data into CSV format.
Tackk

A free service that makes publishing content on the web as easy as writing an email.
TagCrowd

TagCrowd is a web-based word cloud generator that can accept a URL, text, or an uploaded plain text file.
TAPoR (Text Analysis Portal for Research)

The TAPoR Portal is an online environment where users can keep track of texts they want to study (uploaded or available online), learn about and try different tools, and run tools on texts.
TEI Boilerplate

TEI Boilerplate is a lightweight solution for publishing styled TEI (Text Encoding Initiative) P5 content directly in modern browsers. With TEI Boilerplate, TEI XML files can be served directly to the web without server-side processing or translation to HTML.
TemaTres Controlled Vocabulary Server

"TemaTres is an open source vocabulary server, web application to manage and exploit vocabularies, thesauri, taxonomies and formal representations of knowledge." -- TemaTres home page (viewed 30 October 2012)
Tesseract

Combined with the Leptonica Image Processing Library Tesseract can read a wide variety of image formats and convert them to text in over 40 languages.
Tesserae

The Tesserae project aims to provide a flexible and robust web interface for exploring intertextual parallels.
Text 2 Mind Map

Text 2 Mind Map takes a structured list of words or sentences and generates a mind map.
Text Analysis For Me Too (TAToo)

TAToo is an embeddable Flash widget that displays TAPOR analytics for the page on which it resides.
Text Analysis Markup System (TAMS) Analyzer

TAMS Analyzer is a program that works with TAMS to let you assign ethnographic codes to passages of a text just by selecting the relevant text and double clicking the name of the code on a list. It then allows you to extract, analyze, and save coded information.
Text Fixer

Text Fixer allows users to copy and paste a Word document into a box and convert it to clean HTML.
Text Variation Explorer

TVE is an interactive Java tool for exploring the effect of window size on three common linguistic measures: type-token ratio, proportion of hapax legomena, and average word length.
Textal

A free iOS app for text analysis. Textal allows you to analyze documents, tweet streams, and webpages. Create clickable text clouds based on the source data that you choose. It comes pre-loaded with a large number of public domain texts.
Textalyser

An online text analysis tool that provides detailed statistics of your text, including features like the anlysis of words groups, finding out keyword density, analysing the prominence of word or expressions.
TextExpander

TextExpander allows you to customize abbreviations to insert "snippets" for your frequently used text strings or images. This helps you save time typing repeated words or sentences whether on email signatures or several paragraphs of a standard response.
TexTexture

Textexture is a tool for visualizing any text as a network. The resulting graph can be used to get a quick visual summary of the text, read the most relevant excerpts (by clicking on the nodes), and find similar texts.
TextGrid

TextGrid is a virtual research environment (VRE) for the humanities, providing integrated access to specialized tools, services and content, and serving as a long-term archive for research data in the humanities.
TextPad

TextPad is a text editor. It may be used as a basic text editor, web page editor, or as part of a programming IDE.

The Observer XT

Things

Easy to use task management software for Mac, iPad, and iPhone.
Thinkport Annotator

The Annotator allows you to analyze any block of text created by other authors. You may use virtual markers to highlight important passages, questions, thoughts, or add comments.
Tianamo

Tianamo (beta) is an information visualization search engine that displays the search results in a 3D landcape where subtopics are presented topographically as peaks.
TiddlyWiki

TiddlyWiki is a reusable personal web notebook. It allows anyone to create personal hypertext documents that can be published on the Web, and also search and tag content.
TILE (Text-Image Linking Environment)

The Text-Image Linking Environment (TILE) is a web-based tool for creating and editing image-based electronic editions and digital archives of humanities texts.
TileMill

Timeflow

Time Flow is an open-source timeline built to help journalists analyze temporal data. The application offers several view modes--timelime, calendar, list, table--to help explore thousands of data points.
TimeGlider

TimeGlider is a data-driven interactive timeline application built using JavaScript. You can "grab" the timeline and drag it left and right, and zoom in and out to view centuries at a time or just hours.
Timeline JS

TimelineJS can pull in media from different sources. It has built in support for: Twitter, Flickr, Google Maps, YouTube, Vimeo, Dailymotion, Wikipedia, SoundCloud and more media types in the future. Creating one is as easy as filling in a Google spreadsheet or as detailed as JSON.
timemap

timemap is a javascript library that enables you to produce web-based maps (e.g. Google, OpenLayers, Bing) along with a SIMILE timeline. You can load datasets (in JSON, KML, or GeoRSS) onto both the map and the timeline and display on the map only the data shown on the timeline.
TimeRime

TimeRime is a web application which allows people to create, view and compare interactive timelines.
Tinderbox

Tinderbox is a personal content assistant that helps you visualize, analyze, and share your notes. Tinderbox provides maps, timelines, charts, and outlines to help you view and organize your information.
TipCam

TipCam is a screen recording software for Windows, which allows you to record images and audio on your screen, and upload to YouTube.
TitanPad

Collaborators can write and edit a document simultaneously. TitanPad does not currently work with iOS devices like Apple's iPad or iPhone.
Tofu

Tofu reformats text into columns for easier reading.
TokenX

TokenX is a multi-document text visualization, analysis, and play tool designed for the XML document tree.
TokenX

TokenX is a web-based environment for visualizing, analyzing and playing with texts. Options include word clouds, highlighting words, keywords in context, replacing words with bocks, highlighting punctuation and non-words, counting words in context and decontextualized, and substituting words....

Topo.ly

Topo.ly is a free and paid online service for capturing and geocoding spatial data from spreadsheets and creating point, territory and heat maps. It is free for limited use (quite generous) and paid when you need to map significant (we are talking substantial) datasets.
TouchGraph SEO Browser

Trailfire

Trailfire allows you to create "trails" -- or guides -- that link together web sites, add notes, and find content via other people's trails.
Transana

Transcriber

Transana is open source software for the transcription and qualitative analysis of audio and video.
Transcript

Transcription Assistant

Transcription Assistant is a tool that assists in transcription, and incorporates metadata about each image and transcription that may be used to search through an electronic library of transcriptions.
Transcriva

Transcriva is transcription software for the Mac. Whether it's your meeting minutes, interviews, lectures, home movies, dictation, speeches, or your favorite TV show, Transcriva can help you transcribe them all.
Transformer

The Transformer is a tool designed for aligned transcribed linguistic data that can convert data between a variety of formats, as well as organize, search and display the data.
Transformer

Trapit

Trello

Trello is a collaboration tool that organizes your projects into boards. In one glance, Trello tells you what's being worked on, who's working on what, and where something is in a process.
Tumblr

Twapper Keeper

TwitterPad

TXM

TypePad

TypeWright
